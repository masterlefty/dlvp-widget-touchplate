    Electrical Connections
  • Connect the tool to an input ground on the TinyG board
  • Connect the touchplate to the Z-min input on the TinyG board
    • Probe Operations
  • Input conditions: enter values for Feed Rate, Height of Touch Plate, and Z-clearance as noted on the various tabs
  • After running, Z-zero is set to the surface on which the touchplate rests
  • Run after each tool change
    • Exercise caution after subsequence tool change without re-running
    • "Go To Machine Home", or "Go To G5x Zero" commands introduce a chance of crashing the tool into the workbed or workpiece
    General Notes
  • The G38.2 Straight Probe cycle runs in the Machine’s base coordinates and units, (i.e) Absolute coordinate system (G53) and metric units [mm]
  • Prefered method is to run under G21 [mm], but the controller will accept and correctly interpret G20 [inch] for units
  • The G28/G30 Go To Position cycle runs only under machine units G21 [mm]
  • Regression testing has been performed:
    • G38.2
      • Runs under machine coordinates and units
      • Will accept input values in mm or or inch, as defined by system units
      • Ignores G91 (relative coordinate mode)
    • G28/G30
      • Runs under machine coordinates and units
      • Must run under machine units; G21 [mm]
        • The controller does not correctly interpret and convert from inch units when switching to base machine units
      • Accepts G91 (relative coordinate mode) commands, but does not implement correctly
        • The controller reads the absolute value settings for position and adds them incrementally to the current position coordinates
    MCS (G53) -
    - Probing sequence runs from the machine’s current X-Y position
    - Clamp workpiece or stock into machine
    - Locate the moveable touchplate on the desired work surface, and position the tool so it is located above the touchplate
    - During operation; the machine switches to G53 mode, machine (absolute) coordinates
      Notes:
      - This routine does not set the G5x Z-axis to zero(0)
      - Any offset coordinate systems (G54, G55, etc) are readjusted based on the new Machine Z-0 value
      - The G5x Z-axis will not be zero(0), unless G5x is mapped to the machine coordinate system (MCS)
      - This routine is useful if you do not utilize work coordinates (WCS), or if current WCS (ex. G54) is mapped to the Machine’s absolute (MCS) coordinates
      - Once the job(s) is/are complete, consider rehoming the Machine’s Z-axis coordinate (G28.2 z0, if limit switches are installed; or adjust manually with the G28.3 z0, if no limit switches).
    WCS (G5x) Floating -
    - Probing sequence runs from the machine’s current X-Y position
    - Clamp workpiece or stock into machine
    - Locate the moveable touchplate on the desired work surface, and position the tool so it is located above the touchplate
    - Useful when the top of the material is defined as workpiece Z0.
    - Run MCS floating probing sequence
    - Sets the active G5x (machine coordinate system) Z-value to zero
      Notes:
      - Useful if you keep the same tool loaded,
      - or if have a defined X-0, Y-0 setup that is always utilized
      - This routine rewrites the G5x Z0 parameter (G10 L2 Px Zn)
      - If workpiece zero is the bottom of the part, consider utilizing the G92 z-offset option that can be easily turned off after running the job(s).
    WCS (G5x) Fixed -
    - The touchplate is fixed to the work surface of the machine
    - Location is established with from a pre- established G30 location
    - Use "Define G30" to set the fixed position
    - Note: G30 settings are not persistent
    - Useful when the bottom of the material is defined as workpiece Z0.
    - Probing sequence runs from predefined X-Y position
    - Run MCS fixed probing sequence
    - Sets the active G5x (machine coordinate system) Z-value to zero(0).
      Notes:
      - Useful when Z-zero is defined as the bottom of the workpiece (i.e. top of the wasteboard).
      - If workpiece zero is defined to be the top of the stock, add the nominal thickness of the stock to the probe height.
      - This routine rewrites the G5x Z0 parameter (G10 L2 Px Zn)
